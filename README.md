Project: Strategic HRM - PIM Module: End-to-End Requirement & Automation Framework

Role: BA/QA (Hybrid Role)

Tool: Python, Playwright, SQL, Draw.io (Flowchart), BDD (Gherkin).

Content:

    Business Analysis (BA):

        Phân tích nghiệp vụ và biên soạn tài liệu PRD chi tiết cho module Quản lý nhân sự (PIM), đảm bảo các quy tắc về dữ liệu (Data constraints) và điều kiện chấp nhận (Acceptance Criteria).

        Thiết kế luồng nghiệp vụ (Workflow) qua Flowchart (Draw.io), tối ưu hóa trải nghiệm người dùng và xử lý các kịch bản lỗi (Exception handling).

    Quality Assurance (QA):

        Xây dựng Framework kiểm thử tự động từ con số 0 theo mô hình Page Object Model (POM) với Python & Playwright, giúp giảm 80% thời gian kiểm thử hồi quy (Regression test).

        Chuyển đổi yêu cầu nghiệp vụ thành kịch bản kiểm thử BDD (Gherkin), tạo sự thống nhất giữa tài liệu yêu cầu và mã nguồn kiểm thử.

        Thiết lập hệ thống tự động chụp màn hình (Auto-screenshot) khi phát hiện lỗi, hỗ trợ đội ngũ lập trình (Dev) rút ngắn thời gian sửa lỗi (Bug fixing).
